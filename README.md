# Qeepl Booking Autotest

Автотест на Playwright (Python) для проверки флоу бронирования багажа.

### Что проверяет
- Поиск города Москва
- Выбор локации и нажатие "Забронировать"
- Выбор дат в календаре (1–2 января)
- Подтверждение и активация кнопки оплаты
- Клик по кнопке оплаты → открытие модалки регистрации (критерий успеха)

### Запуск
```bash
pip install playwright
playwright install
python qeepl_booking.py
